# Supplier Performance Review
### Lykia Fashion — April 2026
### Confidential — For Internal Use Only

---

## Methodology

- **Data period:** All-time (excl. last 7-14 days delivery lag)
- **Scope:** 57 suppliers with 1,000+ units sold = 793K units, 157K returns (19.8%)
- **Benchmark:** Weighted median return rate per category (resistant to outliers)
- **Flagging criteria:** Return rate must exceed benchmark by **both** >5pp absolute **and** >25% relative
- **Minimum data:** 1,000 units per supplier, 100 per size, 500 per category
- **Robustness checks:** SKU concentration, outlier removal impact, size spread analysis

|---|---|

| Return rate | **19.3%** |
| Category benchmark | 13.7% |

## Structural Finding: XXL Size Issue

| Category | XXL Rate | Category Avg | Deviation |
|---|---|---|---|
| bottoms | 31.1% | 22.0% | **+9.1pp** |
| denim | 34.6% | 27.0% | **+7.6pp** |
| outerwear | 28.7% | 21.5% | **+7.2pp** |
| tops | 19.1% | 15.5% | **+3.6pp** |
| co-ords | 23.8% | 20.2% | **+3.5pp** |

XXL over-indexes on returns in **5 of 7 categories**. This is not a single supplier issue — it's industry-wide. The measurement increments between XL and XXL may not be standardized.

**Recommendation:** Review XXL sizing standards with all suppliers. Consider whether specific product types should carry XXL at all.
